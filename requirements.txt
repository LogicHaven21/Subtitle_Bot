عالی! تا اینجا خیلی خوب پیش رفتی. الان فقط مرحله‌ی **«اتصال»** مونده.

نکته‌ی کلیدی اینجاست: **مرورگر معمولی با مرورگری که ربات می‌تونه کنترلش کنه فرق داره.**
اگر روی شورت‌کات (Shortcut) مرورگر دابل‌کلیک کنی، ربات **نمی‌تونه** بهش وصل بشه مگر اینکه اون شورت‌کات رو «دستکاری» کرده باشی.

این مراحل رو دقیق برو تا همه‌چیز وصل بشه:

### مرحله ۱: تنظیم شورت‌کات مرورگر (حیاتی)

برای اینکه ربات بتونه به مرورگر وصل بشه، باید یک "پورت" باز باشه.

1. روی همون شورت‌کاتی که روی دسکتاپ ساختی **کلیک راست** کن.
2. گزینه **Properties** رو بزن.
3. برو توی تب **Shortcut**.
4. قسمتی داری به اسم **Target**. آخرِ نوشته‌های این کادر، یک فاصله (Space) بزن و دقیقاً این متن رو به تهش اضافه کن:
` --remote-debugging-port=9222 --user-data-dir="C:\selenium\EdgeProfile"`
* **مثال:** اگر قبلاً این بود:
`"C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe"`
* **باید تبدیل بشه به:**
`"C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe" --remote-debugging-port=9222 --user-data-dir="C:\selenium\EdgeProfile"`


5. دکمه **OK** رو بزن.

### مرحله ۲: باز کردن صحیح مرورگر

1. **بسیار مهم:** تمام پنجره‌های باز Edge رو ببند (کامل ببند، حتی اگه توی Taskbar هستن).
2. حالا روی همون شورت‌کات که ویرایش کردی دابل‌کلیک کن.
3. یک مرورگر باز میشه (احتمالاً حس می‌کنی تر و تمیزه و هیچی نداره).
4. توی این مرورگر، ۲ تا تب باز کن و توی هر کدوم برو سایت `aistudio.google.com` یا همون جمنای و لاگین کن. (مدل رو روی Pro چک کن).

### مرحله ۳: نصب کتابخانه‌ها (فقط یک بار)

1. برو توی پوشه‌ای که فایل‌های پایتون (`app.py`, `utils.py`, ...) رو ساختی.
2. توی فضای خالی پوشه، دکمه `Shift` رو نگه دار و **کلیک راست** کن.
3. گزینه **Open PowerShell window here** یا **Open in Terminal** رو بزن.
4. این دستور رو تایپ کن و اینتر بزن تا ابزارها نصب بشن:
```bash
pip install -r requirements.txt

**

### مرحله ۴: روشن کردن ربات

توی همون پنجره سیاه (ترمینال)، این دستور رو بزن:

```bash
python -m streamlit run app.py

```

### مرحله ۵: استفاده از داشبورد

بعد از زدن دستور بالا، خودکار یک تب جدید توی مرورگر باز میشه که داشبورد رباتت هست (شبیه یک سایت).

1. توی منوی سمت چپ (سایدبار)، دکمه‌ای هست به اسم **«بررسی اتصال»**. اون رو بزن.
* اگر نوشت **✅ متصل شد!** یعنی عالیه، ربات مرورگر رو پیدا کرده.
* اگر ارور داد، یعنی مرحله ۱ یا ۲ رو درست نرفتی (احتمالاً مرورگر رو با اون دستور خاص باز نکردی).


2. حالا آدرس پوشه ورودی (که زیرنویس‌های انگلیسی توشه) و خروجی رو بده.
3. دکمه **«شروع عملیات ترجمه»** رو بزن.

**الان برو تست کن و نتیجه رو بگو! منتظرم ببینم اولین زیرنویس چطور ترجمه میشه.**
